# Legacy exploration tools

This folder preserves local sandbox scripts that were used during the Parvathy
PhD report/deck development but are not the primary reproduction entry points.

Some scripts contain local absolute paths from the Codex working machine. They
are committed for provenance and auditability of the analysis process, not as
turnkey command-line interfaces. The reproducible entry points live in the
country-specific folders, `shared/tools`, and `report/tools`.

Primary current entry points include:

- `Vietnam/scripts/geocode_vietnam_170_mapbox.py`
- `Vietnam/scripts/run_vietnam_osm_health_pipeline_batch.py`
- `Vietnam/scripts/run_vietnam_osm_health_approx_pareto.py`
- `Timor_Leste/tools/*`
- `shared/tools/*`
- `report/tools/make_integrated_expansion_figures.py`
