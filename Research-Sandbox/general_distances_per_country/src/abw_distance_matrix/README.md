# abw-distance-matrix

Installable facade for the Analytics for a Better World road-network
distance-matrix pipeline.

The package keeps the current research scripts usable while adding stable
console entry points:

```powershell
abw-distances --help
abw-speed-calibration --help
```

The historical scripts still work:

```powershell
py run_pipeline.py --help
py calibrate_speeds.py --help
```
